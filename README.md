# 🧬 Emergence of a Low-Dimensional Energetic Landscape Governing Calcium Binding in Troponin C

## 📌 Summary

This repository provides a physics-informed computational framework to analyze how mutations modulate calcium-binding energetics in cardiac troponin C (cTnC).

We show that mutation-induced free energy changes can be described using a low-dimensional reaction coordinate integrating structural, electrostatic, and dynamical effects.

---

## 🔑 Key Results

- Structural descriptors: Rp = 0.84  
- Electrostatic descriptors: Rp ≈ 0.70  
- Dynamical descriptors: Rp ≈ 0.81  
- Unified model: Rp ≈ 0.92 (LOOCV)

### Reduced Free Energy Model

ΔG ≈ 0.546·RC − 0.164·RDC − 6.614

---

## 🧠 Conceptual Framework

- Reaction Coordinate (RC): dominant energetic axis  
- Regime Discriminant Coordinate (RDC): orthogonal modulation  

Energetics are organized in a low-dimensional phase space:
- Structural proximity → primary driver  
- Dynamics → secondary modulation  
- Electrostatics → fine-scale tuning
